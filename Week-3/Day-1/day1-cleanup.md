# Week 3 – Day 1 – Task 3 (Cleanup Practice)

## 1. Stop running container
- First check running containers by using `docker ps`
- Stop running container by using `docker stop my-nginx`

## 2. Remove container
- Removed the container by using `docker rm my-nginx`

## 3. Verify container removed
- Verify container removed by using `docker ps -a`

## 4. Remove nginx image
- Remove nginx image by using `docker rmi nginx`
