# Linux Cheat Sheet
Essential commands for DevOps beginners

## File System & Navigation
1. `ls -lh` : List files with human-readable sizes
2. `cd` :  Change current directory
3. `cd ..` : moves you up one level 
4. `cd ~` : Takes you back to your user's home directory.
5. `pwd` : Print current working directory
6. `mkdir -p` : Create directory (and parents if needed) | `mkdir -p notes/week1` 
7. `touch` : Create an empty file | `touch test.txt` 
8. `rm -rf` : Remove files/folders recursively | `rm -rf temp/` 
9. `cp -r` : Copy files or directories | `cp script.sh script_backup.sh` 
10. `mv` : Move or rename a file | `mv old.md new.md` 
11. `cat` : Display file contents | `cat README.md` 
12. `grep` : Search for text within files  
13. `find` : Search for files in a path
14. `chmod +x` : Make a script executable | `chmod +x script.sh`
15. `df -h` : Show human-readable disk usage | `df -h` 
16. `whoami` : Show current logged-in user | `whoami`
17. `history` : List recently used commands | `history`
18. `clear` : Clean the terminal screen | `clear` 
19. `echo` : Print text to screen
20. `date` : Command - Show current date and time date
21. `exit` : Closes your current terminal session or logs.
22. `chmod`: Change permissions
23. `nano*`: Simple command-line text editor
